Session Id

Session id is an id which you can use to get acces to special parts of the API, so get one! 

{
    sessionid_example: "U_2018-08-15t08:37:39.525z"
}

Ping example with sessionid (officialy you don't have to use the sessionid). 

const request = require('request');
request(`http://86.93.252.37:8080/api/v1/ping?sessionid=${sessionid}`, (error, response) => {
    // handle everyting ...
});
