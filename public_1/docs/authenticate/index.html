Authenticate

Authenticate is were you can get a sessionid. You can use this to get access to special parts of the API. 

const request = require('request');
request(`http://86.93.252.37:8080/api/v1/authenticate`, (error, response) => {
    // handle everyting ...
});

The returned response should look simular to this: 

{
    body: {
        sessionid: "<YOUR_SESSION_ID>",
        error: "possible error"
    }
}
